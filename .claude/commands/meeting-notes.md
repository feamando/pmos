# Create Meeting Notes (Pat Format)

Create meeting notes using the NGO "Pat" format.

## Format
```markdown
# [Topic] - [YYYY-MM-DD]

**Attendees:** [Names]

## Notes / Project Updates
- Update 1
  - Sub-detail
- Update 2
- Update 3

## Action Items
- [ ] @Owner - Task description - Due: [Date]
- [ ] @Owner - Task description - Due: [Date]
```

## Instructions
1. Ask for: Topic, Date, Attendees, Key discussion points
2. Structure notes with bullets (hierarchy: main point > sub-bullets)
3. Extract explicit action items with owners
4. Save to appropriate location (usually `Team/` or project folder)
5. Use `YYYY-MM-DD-topic-notes.md` naming

Provide the meeting details and I'll structure the notes.
