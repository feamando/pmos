# Performance Update (Pupdate Style)

Create data-dense performance reports with hypothesis-driven analysis.

## Structure

### 1. Headline
- Total Orders/Conversions + CAC
- Format: `[Metric] ([+/-X% WoW]; [+/-X% YoY])`

### 2. Channel Breakdown
- Meta, Affiliate, CRM, Organic, etc.
- Include raw numbers AND % changes

### 3. Hypothesis / Context
- **Why** did metrics move?
- Link to specific events (promos, launches, seasonality)

### 4. Looking Ahead
- Upcoming tests or launches
- Expected impact on metrics

## Example Format
```markdown
# Pupdate: Week of [Date]

## Headline
- **Orders:** 7,476 (-1% WoW; +172% YoY)
- **CAC:** $42.50 (+5% WoW)

## Channel Performance
| Channel | Orders | WoW | YoY |
|---------|--------|-----|-----|
| Meta | 3,200 | +2% | +150% |
| Affiliate | 2,100 | -5% | +200% |
| CRM | 1,500 | -2% | +180% |

## Why?
- Decline linked to reverting to BAU offer post-July 4th promo
- Affiliate drop due to partner contract renewal gap

## Next Week
- Launching [Test Name] on Meta
- Expected +10% lift in CVR
```
